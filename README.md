Installation
------------
[INSTALL][install]

Cloning
-------
```
$ git clone --recursive https://github.com/SIRLab/VSS-SampleCpp.git
```


Development and Use
-------------------

```
$ make proto
$ make 
$ ./VSS-SampleCpp -c [yellow || blue]
```
